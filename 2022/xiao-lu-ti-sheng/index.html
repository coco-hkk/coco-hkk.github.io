软件快捷键

Windows 11

徽键 WIN 开头：

  1. WIN + A 打开快速设置面板，可设置常用功能：WIFI、蓝牙、飞行模式、夜间模式、专注助手、亮度、音量、辅助功能等。
  2. WIN + B 打开系统托盘，焦点会移到托盘，回车后就能看到隐藏托盘图标。
  3. WIN + C 打开 Microsoft Teams 即时通讯。
  4. WIN + D 快速显示桌面，按一下显示桌面，再按一下恢复。所有 windows 版本通用。
  5. WIN + E 打开资源管理器。通用。
  6. WIN + F 空。
  7. WIN + G 空。
  8. WIN + H 语音听写。
  9. WIN + I 打开所有设置面板。
  10. WIN + J 空。
  11. WIN + K 投屏到其他设备，需要支持 Miracast 的设备。
  12. WIN + L 锁屏。通用。
  13. WIN + M 只能窗口最小化，无法像 WIN + D 还原。此外对一些特殊的对话框无用（如 “运行”）。通用。
  14. WIN + Shift + M 全部窗口最大化。
  15. WIN + N 打开通知面板、日历面板。
  16. WIN + O 空。
  17. WIN + P 投影，修改投影模式。通用。
  18. WIN + Q 一键搜索。
  19. WIN + R 运行。通用。
  20. WIN + S 一键搜索，同 Q。
  21. WIN + T 查看一打开程序缩略图。相当于鼠标悬在任务栏中打开的程序上方，显示程序缩略图。
  22. WIN + U 打开辅助功能设置面板。
  23. WIN + V 打开云剪贴板。同一微软账号可在不同机器上获取剪贴板内容。
  24. WIN + W 打开资讯与兴趣栏，左侧组件。
  25. WIN + X 打开简易开始菜单，相当于在徽键右击鼠标。
  26. WIN + Y 空。
  27. WIN + Z 打开窗口布局。
  28. WIN + . 打开表情符号。
  29. WIN + ; 打开表情符号，同上。
  30. WIN + , 临时切换到桌面，一旦松开就恢复。
  31. WIN + 空格 输入法切换。
  32. WIN + + 打开放大镜。WIN + -/+ 调节大小。
  33. WIN + Esc 关闭放大镜。
  34. WIN + →/↑/←/↓组合 窗口排版。
  35. WIN + TAB 虚拟桌面切换。通过 TAB 切换虚拟桌面，方向键选择桌面或桌面内打开的程序，ALT + Shift + ←/→ 移动桌面。
  36. WIN + Ctrl + D 新建虚拟桌面。
  37. WIN + Ctrl + →/← 切换新桌面。
  38. WIN + PrtScn 一键截屏，保存在 此电脑-图片-屏幕截屏。
  39. WIN + Shfit + S 专业截屏，比 PrtScn 更专业。
  40. WIN + Home 最小化非活动窗口，再按一次可复原。
  41. WIN + [1...9] 打开任务栏项目。
  42. WIN + F1 启用系统帮助。

控制键 Ctrl 开头：

  1. Ctrl + A 全选。
  2. Ctrl + B
  3. Ctrl + D 删除文件。
  4. Ctrl + N 快速创建同级界面。
  5. Ctrl + W 关闭窗口。同 Alt + F4。
  6. Ctrl + Esc 等价于 Win 键。
  7. Ctrl + [1...9] 网页标签页切
  8. Ctrl + Win + D 新建虚拟桌面。
  9. Ctrl + Win + →/← 切换新桌面。
  10. Ctrl + Win + O 打开虚拟键盘。
  11. Ctrl + Shift + Esc 打开任务管理器。
  12. Ctrl + Shift + Delete 打开安全窗口，提供切换用户、注销、锁屏、更改密码、任务管理器功能。

Shift 开头：

  1. Shift + F10 相当于右键展开菜单窗口。
  2. Shift + Delete 永久删除对象。

Alt 开头：

  1. Alt + TAB 打开的程序顺序切换。
  2. Alt + Shift + TAB 打开的程序逆序切换。
  3. Alt + F4 关闭当前窗口，在桌面点击可快速打开关机面板。
  4. Alt + ↓ 打开下拉列表框。
  5. Alt + ← 返回。
  6. Alt + → 前进。
  7. Alt + 空格 打开活动窗口的快捷菜单。
  8. Alt + Esc 按项目打开顺序循环浏览。
  9. Alt + Enter 显示所选项目的属性，等价于 Alt + 双击.

功能键：

  1. F2 重命名文件。
  2. F3 在文件资源管理器中搜索文件或文件夹。
  3. F4 在文件资源管理器中显示地址栏列表。
  4. F5 刷新当前窗口。
  5. F6 循环浏览窗口中或桌面上的屏幕元素。
  6. F10 激活程序菜单栏选项。

复制粘贴：

  1. Ctrl + C 复制，等价与 Ctrl + Insert.
  2. Ctrl + V 粘贴，等价于 Shift + Insert.
  3. Ctrl + X 剪切。
  4. Ctrl + Z 撤销。
  5. Ctrl + Y 恢复操作。

效率提升

  • 为常用软件定义快捷键。
    1. 只有 快捷方式 才可以自定义快捷键。不管是 二进制程序 还是 批处理文件，发送桌面快捷方式。
    2. 打开快捷方式的属性，在快捷键中键入自定义按键即可。
  • 合理组织目录和文件。
  • 合理组织桌面布局。

Chrome 浏览器

书签管理：

  1. Ctrl + D 保存书签
  2. Ctrl + Shift + B 显示书签栏
  3. Ctrl + Shift + O 书签管理器

标签页管理：

  1. Ctrl + T 打开新标签页
  2. Ctrl + W 关闭标签页
  3. Ctrl + Shift + T 重新打开关闭的标签页
  4. Ctrl + Tab 切换到下一个标签页
  5. Ctrl + Shift + Tab 切换到上一个标签页
  6. Ctrl + Shift + A 搜索（最近关闭的）打开的标签页
  7. Ctrl + 鼠标点击链接 在新标签页打开链接

窗口管理：

  1. Ctrl + N 打开新窗口
  2. Ctrl + W 关闭窗口
  3. Ctrl + Shift + N 打开新无痕窗口

导航：

  1. Alt + ← 后退（返回到上一步）
  2. Alt + → 前进

开发者工具：

  1. F2
  2. Ctrl + Shift + I

其它：

  1. Ctrl + + 放大页面
  2. Ctrl + - 缩小页面
  3. Ctrl + E 使用默认搜索引擎搜索
  4. Ctrl + F 查找
  5. Ctrl + H 历史记录
  6. Ctrl + J 下载内容
  7. Ctrl + K 使用默认搜索引擎搜索
  8. Ctrl + L 定位到地址栏
  9. Ctrl + P 打印
  10. Ctrl + R 刷新，等同 F5
  11. Ctrl + S 网页另存为
  12. Ctrl + U 查看网页源代码
  13. Ctrl + Shift + Del 清除浏览数据
  14. Shift + Esc 任务管理器

Total Commander

教程 https://xbeta.info/studytc/index.htm.

配置文件：

  1. wincmd.ini Total Commander 设置文件，保存在 ~\AppData\Roaming\GHISLER\

菜单栏：

  1. Alt + 菜单字母 Alt 键激活菜单键。打开配置是 Alt + O，打开配置->选项时 Alt + O + O

工具栏：

  • 将应用程序拖到工具栏，可以添加程序快捷方式

标签栏：

  1. Ctrl + T 新建标签页
  2. Ctrl + W 关闭标签页
  3. Ctrl + Shift + W 关闭全部未锁定标签页
  4. Ctrl + TAB 切换下一个标签页
  5. Ctrl + Shift + TAB 切换上一个标签页
  6. Ctrl + ↑ 在新标签中打开目录

书签栏：

  1. Ctrl + D 唤出书签栏菜单，打开的内容在另一视图

左右视图：

  1. TAB 切换左右视图
  2. Ctrl + U 调换左右视图
  3. Alt + F1 切换左边视图驱动栏盘符
  4. Alt + F2 切换右边视图驱动栏盘符
  5. Ctrl + →/← 在另一视图打开目录内容，若当前光标在文件则在另一视图打开当前目录内容

显示模式：

  1. Ctrl + F1 列表显示
  2. Ctrl + F2 详细信息显示
  3. Ctrl + Shift + F1 缩略图显示
  4. Ctrl + Shift + F2 文件注释显示
  5. Ctrl + B 展开所有文件

快速搜索：

  1. Ctrl + S 打开快速搜索框，ESC 取消

批量筛选：

  1. <小键盘 +> 批量选中
  2. <小键盘 -> 取消批量选中
  3. <小键盘 *> 反向选择
  4. <小键盘 /> 重选之前的选择
  5. Ctrl + <小键盘 +> 全部选中，Ctrl + A 可替代
  6. Ctrl + <小键盘 -> 取消全选
  7. Alt + <小键盘 +> 选择所有扩展名相同的文件
  8. Alt + <小键盘 -> 取消选择所有扩展名相同的文件
  9. <Space> 选择/取消文件并计算目录大小
  10. Shift + <Space> 选择/取消文件，不计算目录大小

批量重命名：

  1. Ctrl + M 唤出重命名界面（先选中需要重命名的文件）

目录跳转：

  1. Ctrl + \ 跳转到磁盘根目录，如果标签锁定则跳转到标签根目录
  2. Backspace 返回上一层目录

历史路径：

  1. Alt + ⬅ 后退到上一个历史路径
  2. Alt + ➡ 前进到下一个历史路径
  3. Alt + ↓ 显示当前历史路径

命令行：

  1. Ctrl + ↓ 打开命令行历史
  2. Ctrl + E 命令行显示上一次执行的命令
  3. Ctrl + Y 清空命令行
  4. Ctrl + Enter 获取光标所在文件/目录的 名称 并显示在命令行中。然后 Enter 会打开文件或 windows 资源管理器
  5. Ctrl + Shift + Enter 获取光标所在文件/目录的 绝对路径 并显示在命令行中。然后 Enter 会打开文件或 windows 资源管理器
  6. Shift + Enter 在命令行输入 . 然后执行此快捷键，在当前目录打开外部 shell

其它：

  1. Alt + Shift + Enter 计算并显示目录大小
  2. Ctrl + Q 在另一个视图显示文件预览，再按一次关闭预览
  3. Ctrl + Shift + C 复制文件名及完整路径

插件

  • CudaLister 多种语言语法高亮

自定义指令

打开菜单栏 配置 -> 其他 -> 重新定义快捷键 自定义快捷键指令。

  1. Alt + D 编辑文件路径
  2. Alt + R 打开命令终端

其他

  1. 进行编号时，使用字母进行分组，然后使用数字编号，可以提高效率。

    电脑排序 1, 2, 3, 12, 23 开头的文件或目录，会以 1, 12, 2, 23, 3 的方式排列。 先将他们以 a, b, c 分组，即 1, 2, 3 分别为 a1, a2, a3；12 为 b1；23 为 c1。这样会提高效率。

  2. 多台电脑处理数据时，每台电脑需要分组编号。同样，数据也需要分组编号，确保一台电脑所处理的数据不会由其它电脑处理，数据和电脑一一对应。
