介绍一些 vim 实用的插件，本文所介绍插件均以 vim-plug 安装。

vim-plug 插件管理

vim-plug 是一个 vim 插件管理器，利用异步 并行可以快速地安装、更新和卸载插件。

安装

  1. 下载 plug.vim 文件，根据不同操作系统，放置在相应的 autoload 目录中。

    • Linux

      用户配置： ~/.vim/autoload

      全局配置： 安装位置/autolaod

    • Windows

      用户配置： ~\vimfiles\autoload

      全局配置： 安装位置\autoload

  2. 结构目录如下，之后安装的插件将存放在 plugged 目录中。

~/.vim
+----- autoload
|      +-- plug.vim
+----- plugged
       +-- drawit
       +-- ...

注： plugged 目录是由 vimrc 中 call plug#begin() 中定义，可修改。

  1. 在 vimrc 文件中的配置。
call plug#begin('~/.vim/plugged')

" Shorthand notation for plugin
Plug 'vim-scripts/DrawIt'

call plug#end()

插件管理

以 vim 画图插件 DrawIt 为例。

安装插件

vim 可以在 GitHub 上面找到几乎所有的插件。

vim 官网的所有插件都在 https://github.com/vim-scripts/ 中。

vim Awesome 几乎网罗了网络上所有的插件，并提供各种管 理工具的下载方式。

若安装新插件，只需在 call plug#begin() 和 call plug#end() 之间添加插件 github 缩写地址 Plug 'vim-scripts/DrawIt'，然后执行命令:

:PlugInstall

即可安装所有添加插件。

若要安装指定的插件，可执行:

:PlugInstall DrawIt

更新插件

批量更新所有已安装插件：

:PlugUpdate

更新 vim-plug 插件自身：

:PlugUpgrade

插件卸载

若要卸载插件，需要在 vimrc 中注释掉或删除相应的插件，然后执行：

:PlugClean

插件状态

:PlugStatus

fzf 模糊搜索

fzf 全称 Fuzzy finder，由 golang 编写的，用于模糊搜索的工具。结合其他工具可以完 成非常多得工作，如 ag、fasd、ripgrep。

类似工具有 CtrlP、LeaderF。

安装

有很多方式可以安装 fzf，由于 fzf 是独立于 vim 可以运行在各个平台的工具，所以可以独立于 vim 安装。

使用 git 安装

git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install

使用 vim-plug 安装

Plug 'junegunn/fzf', { 'do': { -> fzf#install() } }
Plug 'junegunn/fzf.vim'

我们使用 vim-plug 安装，同时安装插件 fzf.vim。使用 vim-plug 安装方式，我们就可以 通过 :PlugUpdate fzf 指令更新。

配置

" 在当前目录搜索文件
nnoremap <silent> <Leader>f :Files<CR>

"切换Buffer中的文件
nnoremap <silent> <Leader>b :Buffers<CR>

"在当前所有加载的Buffer中搜索包含目标词的所有行，:BLines只在当前Buffer中搜索
nnoremap <silent> <Leader>p :Lines<CR>

"在Vim打开的历史文件中搜索，相当于是在MRU中搜索，:History：命令历史查找
nnoremap <silent> <Leader>h :History<CR>

" 调用 Rg 进行搜索，包含隐藏文件。需要安装 ripgrep 工具，
" https://github.com/BurntSushi/ripgrep
command! -bang -nargs=* Rg
  \ call fzf#vim#grep(
  \   'rg --column --line-number --no-heading --color=always --smart-case --hidden '.shellescape(<q-args>), 1,
  \   <bang>0 ? fzf#vim#with_preview('up:60%')
  \           : fzf#vim#with_preview('right:50%:hidden', '?'),
  \   <bang>0)

主题美化

spaceline 状态栏

安装

Plug 'glepnir/spaceline.vim'
Plug 'ryanoasis/vim-devicons'

配置

配置说明可参考 spaceline。

使用 vim-devicons 需要先 安装字体 并设置 guifont。

set guifont=DaddyTimeMono_Nerd_Font_Mono:h11:cANSI:qDRAFT

let g:spaceline_colorscheme = 'space'
let g:spaceline_seperate_style = 'curve'

airline

安装

Plug 'vim-airline/vim-airline'
Plug 'vim-airline/vim-airline-themes'

配置

配置说明参考 airline。

" vim-airline
let g:airline_powerline_fonts = 1

let g:airline#extensions#tabline#enabled = 1
let g:airline#extensions#tabline#buffer_nr_show = 1

" vim-airline-themes 使用 solarized 主题
let g:airline_theme="solarized"

vim-colorschemes 配色

安装

Plug 'flazz/vim-colorschemes'

配置

set background=light
colorscheme solarized

Markdown

vim-markdown

安装

Plug 'godlygeek/tabular'
Plug 'plasticboy/vim-markdown'

配置

配置说明可参考 vim-markdown。

" 支持 LaTeX 数学语法
let g:vim_markdown_math = 1

vim-markdown-toc 目录

安装

Plug 'mzlogin/vim-markdown-toc'

配置

配置说明可参考 vim-markdown-toc。

" 存储时取消自动更新
let g:vmt_auto_update_on_save = 0

markdown-preview 预览

安装

" 需要安装 nodejs 和 yarn
Plug 'iamcco/markdown-preview.nvim', { 'do': 'cd app && yarn install'  }

配置

同类型的还有 vim-instant-markdown，体验较差。本插件配置说明可参考 markdown-preview。

pangu 排版

安装

Plug 'hotoo/pangu.vim'

配置

配置说明可参考 pangu。

有一些问题：

  • 引用图片时，地址不能包含中文。引用时的方括号一定要在同一行。
