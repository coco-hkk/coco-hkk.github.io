Win10 neovim 安装和配置。

安装

neovim 官网托管在 github 上，从 这里 下载各平台最新版本。

  1. 下载最新版本，例如 0.4.4 nvim-win64.zip。
  2. 将压缩文件解压到适当的位置，例如 D:\neovim，运行 nvim-qt.exe 即可。
  3. 添加环境变量 Path D:\neovim\bin。
  4. 为了方便，可将 nvim-qt.exe 快捷方式发送到桌面。

添加右键 “使用 nvim 打开”

方法一：

  1. 打开注册表 HKEY_CLASSES_ROOT -> * -> shell。
  2. 右键新建项，输入 用 Nvim 编辑。
  3. 在新项中新建 command 项，修改其默认值为 "D:\neovim\bin\nvim-qt.exe" "--fullscreen" "%1"

方法二：

  1. 新建注册表脚本 menu.reg。

    Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\*\shell\Edit with nvim]
"Icon"="D:\\Neovim\\bin\\nvim-qt.exe,0"
@="用 Nvim 编辑"

[HKEY_CLASSES_ROOT\*\shell\Edit with nvim\command]
@="D:\\Neovim\\bin\\nvim-qt.exe  --fullscreen --no-ext-tabline \"%1\""

  2. 双击 menu.reg 导入到注册表中。

    • Icon 所指向的路径是 nvim 程序文件 (exe 程序本身就是一个打包格式，里边包含图标信息）。
    • command 指向的是路径是右键菜单要执行的程序。

配置

win10 中 vim 用户配置。

编辑器 配置文件 配置文件路径 插件路径
vim _vimrc 或 _gvimrc ~/_vimrc 或 _gvimrc ~/.vim 或 ~/vimfiles，可修改
neovim init.vim 或 ginit.vim ~/AppData/Local/nvim 由用户指定

win10 中 vim 全局配置。

编辑器 配置文件 $VIM 配置文件路径 插件路径
vim _vimrc 或 _gvimrc 安装位置 $VIM/_vimrc 或 _gvimrc $VIM/vimfiles
neovim sysinit.vim 安装位置 /share/nvim $VIM/sysinit.vim 由用户指定

PS： 不理解 nvim 为啥要把 init.vim 放在 C 盘隐藏文件中，不方便且乱。

配置文件

可以直接使用 vimrc 的配置。冲突的地方用 if has('nvim') 来适配 nvim。
