Emacs 基本操作

约定

C 表示 Ctrl ，M 表示 Alt ，S 表示 Shift，DEL 表示 Backspace，RET 表示 Enter。

Ctrl-x 或 C-x 表示 Ctrl+x，M-x 表示 Alt+x。

Ctrl-x Ctrl-f 表示先按 Ctrl-x 再按 Ctrl-f。实际操作时按住 Ctrl 然后依次按 x 和 f。

ESC 由 C-[ 替代。

C-g 取消之前的按键。

[操作内容] 指令的操作内容。具体操作内容用文字描述。

Help

;; help 帮助指令
C-h ?    ;; 查看帮助的帮助
C-h m    ;; 查看当前模式
C-h c    ;; 显示快捷键绑定的命令
C-h k    ;; 显示快捷键绑定的命令和它的作用
C-h f    ;; 显示函数的功能
C-h b    ;; 显示当前缓冲区所有可用的快捷键
C-h t    ;; 打开emacs教程
C-h v    ;; 查看变量值
C-h o    ;; 查看符号
C-h w    ;; 查看某个函数绑定的快捷键
C-h r    ;; 查看使用手册

File 管理

C-x C-f    ;; 打开文件，若不存在则创建
C-x C-s    ;; 保存文件
C-x C-w    ;; 另存文件

其它 File 操作

C-x r    ;; 以只读方式打开文件
C-x C-q  ;; 改变窗口只读属性

C-x i    ;; 插入其他文件内容到当前 buffer

C-x s    ;; 保存所有文件

Buffer 管理

C-x b    ;; 切换 buffer，若不存在则创建
C-x C-b  ;; 查看 buffer
C-x C-s  ;; 保存 buffer
C-x k    ;; 关闭 buffer

批量管理 buffer

C-x C-b  ;; 进入 Buffer 列表
d        ;; 标记删除
u        ;; 取消当前行标记
U        ;; 取消全部标记
x        ;; 执行操作
?        ;; 查看按键帮助

Window 管理

C-x 0    ;; 关闭当前 window
C-x 1    ;; 关闭其它 window
C-x 2    ;; 水平分割 window
C-x 3    ;; 垂直分割 window
C-x o    ;; 切换 window

在另一个 window 操作 buffer

C-x 4 b    ;; 在另一个 window 打开 buffer
C-x 4 C-o  ;; 在另一个 window 打开 buffer 但不选中
C-x 4 f    ;; 在另外一个 window 打开文件
C-x 4 d    ;; 在另外一个 window 打开文件夹
C-x 4 0    ;; 关闭当前 window 和 buffer

当一行不能显示所有内容时，该行左右两边会有一个箭头表示前后还有内容

C-x <    ;; 看右边的东西
C-x >    ;; 看左边的东西

Tab 标签管理

C-x t 2    ;; 新建 Tab
      1    ;; 关闭其它 Tab
      0    ;; 关闭当前 Tab
      b    ;; 在新 Tab 中打开 Buffer

Frame 管理

C-x 5 2    ;; 创建 frame
C-x 5 b    ;; 在新 frame 中打开 buffer
C-x 5 o    ;; 切换 frame
C-x 5 0    ;; 删除 frame
C-x 5 f    ;; 在新 frame 中打开或创建文件
C-x 5 b    ;; 在新 frame 中打开或创建 buffer

Dired 管理

打开文件管理 C-x d 或 M-x dired-jump 或 C-x C-f。

导航

  • RET 打开目录或文件
  • S-RET 在另外一个 window 中打开文件
  • M-RET 在另外一个 window 中预览文件
  • g o 以预览方式打开一个文件，q 关闭
  • n 下一项
  • p 上一项
  • j 在 buffer 中跳转文件
  • ^ 返回父级目录
  • q 退出 Dired buffer

标记文件

  • m 标记光标指定文件
  • u 取消标记的文件
  • U 取消所有标记的文件
  • * t 反选 buffer 中未标记文件
  • * 其它自动标记函数
  • % m 使用正则表达式标记文件
  • k 删除所有 mark 标记，需使用 g 刷新

复制和重命名文件

  • C 复制文件
  • R 重命名文件
  • % R 使用正则表达式重命名

删除文件

  • D 删除文件
  • d 标记待删除文件
  • x 确认并执行删除已标识删除的文件

压缩和解压文件

  • Z 压缩或解压格式为 .tar.gz 的文件或目录
  • c 选择压缩指定文件

属性和权限

  • O 变更文件 owner
  • G 变更文件所属组
  • M 变更文件权限

其它

  • g 刷新，重新加载已修改文件
  • T 修改时间戳
  • + 创建新目录
  • S 创建符号链接文件
  • L 加载 emacs lisp 文件到 emacs
  • ~ 给所有的 backup 文件设置标识
  • = 比较文件

剪切复制粘贴

  1. C-w 剪切

  2. M-w 复制

  3. C-y 粘贴

  4. 切换粘贴内容 M-y，在按下 C-y 后执行

    操作 按键
    剪切 C-w
    复制 M-w
    粘贴 C-y
    切换粘贴内容 M-y

  5. M-<DEL> 剪切当前光标前的一个字符到剪贴板

  6. M-d 剪切当前光标后的一个字符到剪贴板

  7. C-k 剪切从光标到行尾的字符到剪贴板

  8. C-a C-k 剪切一整行到剪贴板中，其实也可以用于删除一整行

  1. 使用 Ctrl c, Ctrl v, Ctrl x 标准快捷键
    • 方法一 M-x cua-mode
    • 方法二修改配置文件，增加 (cua-mode 1)

历史命令

  • M-p 上翻历史命令
  • M-n 下翻历史命令

书签

  • C-x r m 在当前光标设置书签(M-x bookmark-set)
  • C-x r b 跳转到某个书签(M-x bookmark-jump)
  • C-x r l 列出当前所有标签(M-x bookmark-bmenu-list)

在列表书签中

  • RET 打开书签标识的区域
  • s 将当前书签列表保存至文件中
  • r 修改书签名称
  • d 设置删除标识
  • x 删除标识书签
  • u 取消标识书签

文本操作

移动

  1. C-n 将光标向下移动一行
  2. C-p 将光标向上移动一行
  1. C-b 将光标左移一个字符
  2. C-f 将光标右移一个字符
  1. M-f 将光标右移一个单词
  2. M-b 将光标左移一个单词
  1. C-v 向下移动一屏
  2. M-v 向上移动一屏
  3. C-l 将当前光标所在行移动到屏幕中央（注意：这里是字母L）
  1. C-a 将光标移到行首
  2. C-e 将光标移到行尾
  3. M-m 将光标移动到行首的第一个非空白字符
  1. M-S-< 将光标移动到文件开头
  2. M-S-> 将光标移动到文件末尾

文本选择

  1. C-S-e 选中从当前位置到行尾的文本
  1. C-S-n 从当前位置开始往下选中一行文本
  2. C-S-p 从当前位置开始往上选中一行文本
  1. C-S-f 从当前位置开始往后选中一个字符
  2. C-S-b 从当前位置开始往前选中一个字符
  1. M-S-f 从当前位置开始往后选中一个单词
  2. M-S-b 从当前位置开始往前选中一个单词
  1. M-S-e 选中从当前位置开始到当前句尾的文本
  2. M-S-a 选中从当前位置开始到当前句首的文本
  1. C-S-v 向下选择一屏
  2. M-S-v 向上选择一屏
  1. C-S-l 让当前光标所在行居中
  2. C-x h 全选
  3. C-S-@ [其他移动操作] 选中从光标起始位置到移动后位置中间的文本

文本编辑

  • C-u [N] [操作序列] 重复执行一个操作序列 N 次
  • C-/ 或 C-x u 撤销（undo）上次操作，多次按可以撤销之前的多次操作
  • C-g C-/ 重做（redo），用于重做上次的撤销动作（撤销之后后悔了，想恢复重做）
  • C-k 删除光标到行尾的所有字符
  • <DEL> 删除当前光标前的一个字符
  • C-<DEL> 删除当前光标前的一个单词
  • C-d 或 <Delete> 删除当前光标后的一个字符
  • C-e <RET> 在当前行下方插入一空行
  • C-x C-q 让当前buffer在“只读”和“读写”两种编辑模式之间切换
  • M-\ 删除光标处所有空格和 TAB。
  • M-SPC 删除光标处所有空格和 TAB，但保留一个空格。
  • C-x C-o 删除光标周围的空白行，保留当前行。
  • C-t 交换两个字符的位置。
  • M-t 交换两个 word 的位置。
  • C-x C-t 交换两行的位置。
  • M-/ 匹配这个首字母的单词。
  • M-c 将下一个单词开头大写，后面小写。

文本字数统计

;; 整个 buffer 字数统计
M-=
;; 或者
M-x count-words-region

;; 选中区域统计
M-x count-words

查找与替换

  • C-s [查找的内容] 向下/右查找
  • C-r [查找的内容] 反向查找
  • C-M-s [查找的内容] 向下/右正则查找
  • C-M-r [查找的内容] 反向正则查找
  • M-% 查找并替换。输入 y 替换并跳到下一个，输入 n 忽略并跳到下一 个，输入 q 结束，输入 ！ 替换剩下的全部。
  • M-s o 正则匹配查找，并显示匹配列表。
  • M-s h . 查找光标所在 word，并高亮显示。

文本编码

  1. M-x describe-coding-system <RET> 查看当前缓冲区的编码格式
  2. C-x <RET> r <TAB> 列出当前的所有编码
  3. C-x <RET> f utf-8 <RET> 改变当前缓冲区的编码为 utf-8

杂项

  • C-q Tab 输入 TAB
  • M-g M-g [行号] 跳转到指定行数
  • M-x untabify 转换 TAB 为空格。先选中转换范围，然后执行，如选中整篇文档 C-x h。
  • C-M-\ 自动缩进
  • C-x f 指定填充列，可设置为 80 换行。

编程

  • C-c . 程序缩进风格
  • C-M-\ 自动缩进

游戏

  • M-x hanoi 汉诺塔，默认 3 个碟子。M-x hanoi-unix 和 M-x hanoi-unix-64 使用 unix 时间戳的位数作为默认碟子个数，并且每秒钟自动移动一次。
  • M-x 5x5 5x5 逻辑解密游戏，你有一个 5x5 的网格，中间的十字被填满；你的目标是通过按正确的顺序切换它们的空满状态来填充所有的单元格，从而获得胜利。
  • M-x animation-birthday-present 文本动画。
  • M-x butterfly 面向 XKCD 漫画致敬。
  • M-x blackbox 黑箱。游戏的目标是通过向黑盒子发射光线来找到四个隐藏的球。有四种可能：1) 射线将通过盒子不受干扰；2) 它将击中一个球并被吸收；3) 它将偏转并退出盒子，或 4) 立即偏转，甚至不能进入盒子。
  • M-x bubble 泡泡。你必须用尽可能少移动清除尽可能多的“泡泡”。当你移除气泡时，其他气泡会掉落并粘在一起。
  • M-x doctor 心理医生。
  • M-x dunnet 文字冒险游戏。
  • M-x gomoku 五子棋。
  • M-x life 生命游戏。
  • M-x tetris 俄罗斯方块。
  • M-x pong 乒乓。
  • M-x snake 贪吃蛇。
  • M-x solitire 一个基于“钉子”的游戏，你可以选择一块石头（o）并“跳过”相邻的石头进入洞中（.），并在这个过程中去掉你跳过的石头，最终只能在棋盘上留下一块石头，重复该过程直到棋盘被请空（只保留一个石头）
  • M-x zone 屏幕保护程序。M-x zone-leave-me-alone 关闭，M-x zone-when-idle 配置屏幕保护程序的空闲时间。
  • M-x mpuz 乘法解谜题。M-x mpuz-show-solution 查看答案。
  • M-x morse-region 和 M-x morse-region 将一个区域（英文）翻译成摩尔斯电码。

其它

ctrl space 设置 mark

由于 ctrl space 被输入法占用，emacs 每次 mark 时得 ctrl @很麻烦，所以想关闭 win10 输入法热键。但是试了多种方法，没有好的解决方法。只能先将输入法切换到微软自 带的 ENG 模式，ctrl + 空格 才有效。

同时在选中的各行行首添加相同内容

  1. 将光标置于第一行行首。
  2. ctrl space (set-mark-command) 设置标记。
  3. 将光标置于最后一行行首。
  4. 执行 C-x r t (M-x string-rectangle)。
  5. 输入任何想输入内容，然后回车。

寄存器

  1. 存储到寄存器

    选择文本，执行C-x r s (M-x copy-to-register)，输入 1，可将文本保存在了名 为 1 的寄存器。

  2. 从寄存器粘贴

    执行 C-x r i (M-x insert-register)，输入 1，可将名为 1 的寄存器内容粘贴 到光标所在位置。

  3. 寄存器名

    单数字 0-9，或单字符 a-z。

色号

色号 M-x list-colors-display。

emacs 自带输入法

自带输入法 C-\。

shell 命令

M-! 输入 shell 命令。
